# Mobs Tactician

Minecraft NeoForge mod

让敌对生物拥有更战术化、更接近玩家思路的战斗行为。

> 目前为实验预览版本，不按原版生存平衡设计。

## 主要内容

| 生物 | 主要改动 | 效果 |
| --- | --- | --- |
| 骷髅 | 战术远近战 AI | 搭掩体、抓盾牌、切换武器 |
| 僵尸 | 风弹重锤攻击 | 跳跃接近并发动空中重锤攻击 |
| 幻翼 | 火箭俯冲 | 使用烟花火箭加速俯冲 |
| 苦力怕 | 预测式膨胀 | 在玩家明显近战冲刺时提前引爆 |

## 战术骷髅

强化骷髅可以生成时携带附魔弓、备用近战武器、防御方块，有时还会穿戴护甲。

+ 玩家举盾时，骷髅会保持拉弓，等待盾牌放下后再射击
+ 玩家即将用弓射击它时，会快速放置一小段泥土/圆石掩体
+ 玩家近身时切换到剑
+ 距离拉开后切回弓
+ 近战中被连续压制时，会短暂强制切回远程
+ 死亡时可能掉落武器、护甲或剩余防御方块

装备等级：

+ 铁级：弓、铁剑、泥土方块，可能穿铁甲
+ 钻石级：更强的弓、钻石剑、圆石方块、钻石甲
+ 下界合金级：更强的弓、下界合金剑、更多圆石方块、下界合金甲

## 战术僵尸

部分僵尸可以生成时携带重锤和风弹。

+ 使用风弹跳向目标
+ 在下落时尝试发动空中重锤攻击
+ 铁、钻石、下界合金变体拥有不同装备强度
+ 下界合金变体可能获得风爆等更强重锤附魔
+ 幼年鸡骑士僵尸不会获得强化
+ 死亡时可能掉落剩余风弹、重锤或护甲

## 强化幻翼

强化幻翼会携带武器和烟花火箭。

+ 俯冲攻击时使用烟花火箭
+ 游戏难度越高，火箭加速越强
+ 会渲染手持武器和烟花
+ 仍然会被猫吓退
+ 死亡时可能掉落剩余烟花或武器

## 更聪明的苦力怕

苦力怕使用替换后的膨胀目标。

+ 玩家手持近战武器、看向苦力怕并冲向爆炸范围时，苦力怕会提前开始膨胀
+ 难度越高，预测越激进
+ 有目标的苦力怕若将被摔死，会直接爆炸

## 安装

需求：

+ Minecraft `1.21.11`
+ NeoForge `21.11.38-beta` 或兼容版本
+ Java `21`

将构建得到的 `.jar` 放入对应 NeoForge 实例的 `mods` 文件夹。

## 构建

```powershell
.\gradlew.bat build
```

生成的 jar 位于 `build/libs`。

## 说明

+ 本 Mod 有意让部分怪物变得不公平。
+ 更适合挑战整合包、自定义地图或短时间混乱娱乐玩法。
+ 生成概率和行为开关尚未完全配置化。
